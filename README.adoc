Per the link:.claude/CLAUDE.md[language policy], implementation will use:

* *Primary Language:* ReScript (compiling to JS) or Rust
* *Configuration:* Nickel / Guile Scheme
* *Package Management:* Guix (primary), Nix (fallback)
